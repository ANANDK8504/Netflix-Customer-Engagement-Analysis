# Netflix Customer Engagement & Content Analytics (Power BI)

## Project Overview
This project is a multi-page Power BI report created to analyze Netflix customer engagement, content performance, subscription trends, and regional insights. The analysis is based on task-driven business questions and presented using interactive dashboards.

## Business Objectives
- Analyze customer engagement based on viewing behavior
- Compare Movies vs TV Shows performance
- Understand subscription market share and dominance
- Analyze content ratings and director influence
- Identify optimal content release periods
- Study genre-wise and country-wise content distribution
- Evaluate customer retention patterns

## Report Structure
The Power BI report contains multiple pages, each focusing on specific insights:
- Customer engagement overview with KPIs and trends
- Ratings and viewing behavior analysis
- Content release and trend analysis
- Subscription and market share analysis
- Regional and country-wise content insights

## Tools & Technologies
- Power BI
- DAX
- Data Modeling
- Data Visualization

## Key Insights
- TV Shows generally have higher average ratings compared to Movies.
- Premium and Standard subscriptions dominate across most regions.
- Certain genres and directors consistently receive higher ratings.
- Content release trends indicate specific months as optimal release periods.
- Netflix content shows strong global presence with regional variation.
